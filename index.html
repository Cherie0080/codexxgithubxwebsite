Slide 07

Step 1，先註冊 GitHub 帳號

  • 打開 github.com
  • 點 Sign up
  • 輸入 Email、密碼、帳號名稱
  • 完成驗證並登入
提醒：帳號名稱之後會出現在公開網址裡。

Slide 09

Step 3，建立首頁檔案

首頁通常叫 index.html，系統會把它當作網站首頁。

檔名一定要正確，不要存成 index.html.txt。

index.html

Slide 10

最基本的 HTML 首頁範例

<!DOCTYPE html> <html lang="zh-Hant"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>我的第一個公開網頁</title> </head> <body> <h1>歡迎來到我的公開網頁</h1> <p>如果你看到這一頁，代表網站建置成功。</p> </body> </html>
Slide 11

Step 4，把首頁上傳到 GitHub

  • 打開你的 repo
  • 點 Add file
  • 選 Upload files
  • 上傳 index.html
  • 點 Commit changes

Add file

Upload files

拖入 index.html

Slide 12

Step 5，把 repo 變成公開網站

打開 Settings

找到 Pages

選 Deploy from a branch

選 main / root

重點：沒開 GitHub Pages，就不會有公開網址。

Slide 15

常見錯誤與排查

常見錯誤

  • 檔名不是 index.html
  • 存成 .txt
  • repo 沒設 Public
  • Pages 沒開

正確排查

  • 確認首頁檔名
  • 確認 repo 可公開
  • 確認 Pages 設定
  • 等待幾分鐘再測試
