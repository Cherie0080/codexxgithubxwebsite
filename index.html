補充說明：這一步的意義，是把本機上的內容正式放到 GitHub 上，讓網站從自己的電腦，走向可被發布的狀態。

補充說明：真正讓網站公開的不是把檔案上傳而已，而是要把 GitHub Pages 正確打開，這是上線的關鍵開關。
